# Homebrew Tap

Personal Homebrew tap for @HamedMP's projects.

## Installation

```bash
brew tap hamedmp/tap
```

## Formulas

| Formula | Description |
|---------|-------------|
| [idag](https://github.com/HamedMP/idag) | Modern CLI to query macOS Calendar from the terminal |

## Usage

```bash
# Install idag
brew install hamedmp/tap/idag

# Or after tapping:
brew tap hamedmp/tap
brew install idag
```
